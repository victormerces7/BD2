# BD2
Atividade Avaliativa 2

* Atividade Prática em Grupo

1. Tarefas:
    * Utilizando o banco de dados criado na Avaliação 1 (opcional), realizar as seguintes atividades:
        * Popular as tabelas com um volume significativo de dados para simular um ambiente real (pode-se usar scripts para inserir dados ou importar de arquivos).
        * Identificar consultas SQL que apresentem baixo desempenho.
        * Aplicar técnicas de otimização, como:
            * Ajuste e criação de índices.
            * Reescrita de consultas para formas mais eficientes.
            * Análise de planos de execução das consultas.
        * Avaliar o impacto das otimizações realizadas por meio de métricas de desempenho (tempo de execução, consumo de recursos).
        * Efetuar a criação de Views, Triggers e Stored Procedures conforme a necessidade do projeto.
2. Entrega:
    * Apresentação em grupo contendo:
        * Descrição das consultas problemáticas e análise dos motivos do baixo desempenho.
        * Detalhamento das otimizações aplicadas.
        * Comparação dos resultados antes e depois das otimizações.
        * Explicação dos índices, Views, Triggers e Stored Procedures criadas.
